# 人效计算工具

从飞书表格抓取作业数据，管理评估人员名单，自动计算人效并导出看板。

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

浏览器自动打开 http://127.0.0.1:5000

## 功能

| 页面 | 说明 |
|------|------|
| 快照管理 | 粘贴飞书表格/知识库链接，一键抓取作业完成数据（自动解析合并单元格） |
| 人员名单 | 管理需计算人效的人员，设置姓名、公司、默认工时（默认 8h） |
| 填写工时 | 独立设置每位人员的工时，与快照解耦 |
| 人效看板 | 选择两个快照自动计算人效（件/小时），支持导出到飞书表格 |

## 首次使用

1. 在飞书开放平台创建自建应用（企业自建应用 → 获取 App ID 和 App Secret）
2. 将应用添加为目标飞书表格的协作者
3. 启动应用后，按提示配置 App ID 和 App Secret

## 技术栈

Flask + SQLite，飞书 API 直连（无需 lark-cli）。

## 打包为 exe

```bash
pyinstaller 人效计算.spec
```
生成的 exe 在 `dist/` 目录下，发给朋友即可运行。

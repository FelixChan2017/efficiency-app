# 人效计算工具

从飞书表格抓取作业数据，管理评估人员名单，自动计算人效。

## 技术栈

Flask + SQLite + pywebview + lark-cli

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

打开 http://127.0.0.1:5000

## 功能

- **快照管理** — 粘贴飞书链接抓取作业完成数据，自动解析合并单元格
- **人员名单** — 管理需计算人效的人员（姓名、公司、默认工时）
- **填写工时** — 为名单内人员设定工时，默认 8 小时
- **人效看板** — 自动对比最近两次快照，结合工时计算出人效（件/小时），支持导出飞书

## 依赖

- lark-cli（飞书命令行工具，需预先安装登录）
- Python 3.x
